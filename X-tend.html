X-tend
Playable in browser

X-tend was my first real Unity project. This game was made for a hackathon with a team of three people, so there was a real time crunch. As a result, we didn;t get to fix some of the obvious bugs like glitching out of bounds. Besides that, though, the most disappointing thing about this game is that we spent a lot of work getting different features to function properly, but didn't get a chance to add new levels showcasing them working with each other. The puzzle solving is extremely limited and I feel like we could have made much more content with minimal effort.
